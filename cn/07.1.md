# 07.1 比较运算符详解($eq,$gt,$lt,$ne,$in,$nin)

## mongodb具体运算符
```
比较运算符
$eq  =
$gt  >
$gte >=
$lt  <
$lte <=
$ne  !=
$in  in
$nin not in
```

## eq 相等
```
字符创匹配
db.inventory.find({"qty":"abcdefg"});

数组匹配
db.inventory.find({"tags":{$eq:"A"}})  //forech查找
db.inventory.find({"tags":{$in:["A"]}});  //如果采用in的话，我们会更加的简洁。
```

## gt gte 大于 大于等于
```
db.inventory.find( { qty: { $gte: 20 } } )
```

## $lt/$lte 小于 小于等于
```
db.inventory.find( { qty: { $lte: 10 } } )
```

## $ne 不等于
```
db.inventory.find( { qty: { $ne: 10 } } )
```

## $in $nin
db.test123.update({ tags: { $in:["A","E"]} },{$set: { qty:100000000 }})
db.test123.find( { qty: { $nin: [ 5, 20 ] } } )
